# CollegeSpaceV2
Android Platform target CollegeSpacev2 App.

### Minimum Requirements
1. For Users
  * **Android 4.0.0** (Ice Cream Sandwich) or **Later**
2. For Developer
  * Android Studio 2.0
  * API 16 (**21 Recommended**)
  * Google Play Services
